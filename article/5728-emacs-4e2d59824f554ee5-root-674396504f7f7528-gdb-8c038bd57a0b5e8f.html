在emacs中如何以root权限使用gdb调试程序

  • 由于M-x命令中使用sudo输入密码无效，需要配置为允许用户sudo gdb免密码 
visudo
# Allow user to sudo gdb without password
用户 ALL=NOPASSWD: /usr/bin/gdb
  • 使用root权限启动gdb 
M-x gdb
sudo gdb <program> <pid> --annotate=3
