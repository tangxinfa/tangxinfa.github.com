编译安装node.js

  • 下载

    node.js是一个新兴的开发平台，版本更新非常活跃，因此应该尽量下载安装最新的版本。

    到 http://nodejs.org/ 下载最新的版本。

  • 安装

    解压后参照 README.md 进行安装： 

    ./configure
make
make install
  • 安装后

    默认安装到/usr/local。

    /usr/local/bin/node
    node主程序
    /usr/local/bin/npm
    node模块管理程序
    /usr/local/lib/node_modules
    node全局模块目录

    像一些需要全局安装的模块也会把文件安装在/usr/local目录下。
