程序员的书单

虽然下面所列的部分书籍中文译本的质量也非常高，有兴趣可以尝试阅读英文原版，因为最 新的技术资料基本上都是英文的，培养英文阅读能力至关重要。

编程开发基础

  • 《代码大全》

如果你不清楚什么代码是好的，而只是一味只知道模仿同事的代码，那就一定不要错过代码 大全。

系统开发基础

  • 《UNIX 环境高级编程》

世界在变，而 C 和 UNIX（LINUX）却一直是软件世界的基石。

应用开发语言

精通一门应用开发语言。

Go

  • 《The Go Programming Language》
  • 《Concurrency in Go》

Node.js

  • 《Node.js Documentation》

Rust

  • 《Rust by Example》

C++

  • 《Effective C++》
  • 《Effective Modern C++》
