Archlinux 下搭建 Rust 开发环境

  • Emacs 里安装 rust-mode

    M-x el-get-install rust-mode

  • Archlinux 安装 rust 相关包

    yaourt -S rust cargo

    安装的版本 

    $ rustc --version
rustc 1.6.0
$ cargo --version
cargo 0.8.0 (28a0cbb 2016-01-17)

  • Hello world!

    hello_world.rs 

    fn main() {
    println!("Hello, world!");
}

    编译运行 

    $ rustc hello_world.rs 
$ ./hello_world 
Hello, world!

  • 更多配置

    可以参考文章《Configuring Emacs for Rust》进行更高级的配置，对于我这种 Rust 还没入门的人来说，前面的配置已经足够，还是一步一个脚印吧。
